哈希表篇（数组、set、map）

判断元素是否在某集合中出现过 → hash表

242.有效的字母异位词

t 是 s 的异位词等价于「两个字符串排序后相等」。

  • 排序

    将两个字符串转换成数组，然后排序，最后在转换成字符串，比较这两个字符串是否相等即可。

  • 哈希表

    由于字符串只包含 26 个小写字母，因此可以维护一个长度为 26 的频次数组 table，先遍历记录字符串 s 中字符出现的频次，然后遍历字符串 t，减去 table 中对应的频次，如果出现 table[i]<0，则说明 t 包含一个不在 s 中的额外字符，返回 false 即可。

349. 两个数组的交集

思路：

首先确保nums1>nums2,然后将 nums1 转换成一个 Set 集合，遍历nums2，
逐个检查 nums2 中的元素是否存在于 Set 集合中，若存在，则添加到结果 Set 集合中，
最后将结果 Set 集合转化为数组（Array.from()）

202. 快乐数

示例：非快乐数：20

  • 1 20——4+0 = 4
  • 2 4 ——16 = 16
  • 3 16——1+36 = 37
  • 4 37——9+49 = 58
  • 5 58——25+64 = 89
  • 6 89——64+81 = 145
  • 7 145—-1+16+25 = 42
  • 8 42——16+4=20 = 20

Set是es6新增的数据结构，类似于数组，但是成员的值都是唯一的，没有重复的值，我们一般称为集合

  • set方法：

将快乐数转化成字符串，while(sum !== 1)内遍历字符串，得到新Sum值，
若结果 Set 集合内含有Sum值，则非快乐数，反之，则将sum添加进set集合内
最后在while循环外返回true

Map类型是键值对的有序列表，而键和值都可以是任意类型

  • map方法：

初始化一个 Map 用于记录已经出现过的数字 ，定义一个函数 getSum，计算数字平方和
进入无限循环(while(n !== 1))：若 n 已经出现在 Map 中，则表示陷入了无限循环，返回 false;
反之，将n 记录到 Map 中,并计算当前数字各个位数字的平方和getSum(n)，并赋值给 n，进行下一次循环。

最后，若当前数字 n 为 1，返回 true。

1.两数之和

示例： nums = [2, 7, 11, 15], target = 9
返回[0,1]

  • 暴力解法：两层for循环（i=0 j=i+1 nums[i] + nums[j] === target），不推荐

    思路：

遍历当前数组元素，若hash表内匹配到了target-nums[i]，则返回[i, target-nums[i]],
反之，就把当前访问的元素和下标加入到map中

454.四数相加II

示例： A = [1, 2] B = [-2, -1] C = [-1, 2] D = [ 0, 2] 输出：2
(0, 0, 0, 1) -> A[0] + B[0] + C[0] + D[1] = 1 + (-2) + (-1) + 2 = 0
(1, 1, 0, 0) -> A[1] + B[1] + C[0] + D[0] = 2 + (-1) + (-1) + 0 = 0

思路：

初始化哈希表（twoSumMap）和计数器（count）
两层for循环A和B，并记录各个A+B值出现的次数
两层for循环C和D，若能找到0-(C+D),将 twoSumMap 中对应的值（出现次数）加到 count 中。

383. 赎金信

给定一个赎金信 (ransom) 字符串和一个杂志(magazine)字符串，判断第一个字符串 ransom 能不能由第二个字符串 magazines 里面的字符构成。如果可以构成，返回 true ；否则返回 false。

思路：

定义一个长度为26的数组strArr，初始化为0。base 是字符 ‘a’ 的 Unicode 编码。
遍历magazine字符串，记录每个字符出现次数，再遍历ransomNote字符串，如果对应索引的值为0，
表示 magazine 中该字符数量不足，返回 false。否则，对应索引的值减 1。

15. 三数之和

示例：nums = [-1, 0, 1, 2, -1, -4]，
满足要求的三元组集合为：[[-1, 0, 1], [-1, -1, 2]]

思路：

首先，对数组进行排序，遍历; 再建立前后指针 left=i+1 和right=nums.length-1，计算三数之和 sum;
若sum>0,则right向前移动；若sum<0，则left向后移动；sum=0时，将这3个数组成的数组保存进res中。
设置去重：如果当前元素与上一个元素相同，跳过当前循环以避免重复结果。
若left指向的元素与下一个相同，右移左指针；若right指向的元素与上一个相同，左移右指针。

  • nSum通解：递归

18. 四数之和

示例： nums = [1, 0, -1, 0, -2, 2]，和 target = 0
满足要求的四元组集合为： [ [-1, 0, 0, 1], [-2, -1, 1, 2], [-2, 0, 0, 2] ]

思路：

对比三数之和，再套一层for循环（外层循环i 从 0 遍历到 len-4，内层循环j 从 i+1 遍历到 len-3）
注意去重；当数组长度小于 4，直接返回空数组。
